# Contributing to ClawMemory

Thank you for your interest in contributing! We welcome community contributions and feedback.

## How to Contribute

1. Fork the repository.
2. Create a feature branch: `git checkout -b feature/my-enhancement`.
3. Make your changes and add tests if applicable.
4. Commit your changes: `git commit -m "Description of change"`.
5. Push to your fork: `git push origin feature/my-enhancement`.
6. Open a Pull Request against `main`.

## Code Style

- Follow PEP8 conventions for Python code.
- Use `black` for formatting and `flake8` for linting.

## Issue Reporting

Use GitHub Issues to report bugs or request features. Please include:
- A clear description of the problem
- Steps to reproduce
- Expected vs actual behavior

## Testing

Unit tests are welcome. Add tests under a `tests/` directory and configure CI accordingly.
